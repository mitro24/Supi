# Supi


main.py contains the backend of the website, which creates JSON files that contain the database.

The database was created by randomly selecting a number of products from the e-katanalotis API, keeping only what was necessary for our AI helper, Supi, to work with. An example of this database, is the JSON file products_grouped_example.json

ekatanalotis_data.json contains EVERYTHING inside the e-katanalotis API, there was no real reason including this but it's just for the comparison of the two databases
